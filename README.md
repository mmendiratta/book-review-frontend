Here's a blog post with my journey for this project: https://mannimendiratta.medium.com/gifting-a-software-project-7e123fb0b6a6

Quick Synopsis:
- This repo is for the user interface of the book review website
  - It is written with Javascript/React and uses the AntD component library
  - The frontend is hosted with Netlify and the domain is registered/linked through Google Domains
- Here is the repo for the backend/api service of the website (https://github.com/mmendiratta/book-reviews-backend)
  - The API is written in NodeJs using Express
  - The database is MongoDB
